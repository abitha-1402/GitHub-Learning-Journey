# Repository Creation

## What is a Repository?

A repository (or repo) is a place where a project's files and folders are stored. It helps you organize your project and track all changes using Git.

A repository can be stored:
- Locally (on your computer)
- Remotely (on GitHub)


## Why Create a Repository?

- To organize project files.
- To track changes in the project.
- To save project history.
- To collaborate with other developers.
- To store projects on GitHub.

---

## Steps to Create a Repository on GitHub

1. Sign in to your GitHub account.
2. Click the **New Repository** button.
3. Enter a repository name.
4. Add a description (optional).
5. Choose **Public** or **Private**.
6. Click **Create Repository**.

---

## Example

Repository Name:

```text
Git-Learning-Journey
```

Description:

```text
A repository to document my Git and GitHub learning journey.
```

Visibility:

```text
Public
```

---

## Output

After creating the repository, GitHub creates a new repository with a unique URL.

Example:

```text
https://github.com/username/Git-Learning-Journey
```

You can use this URL to clone or connect your local project.


## Common Error

```text
Repository not found
```

**Reason**

The repository URL is incorrect or you do not have permission to access it.

**Solution**

- Check the repository URL.
- Make sure you have access to the repository.

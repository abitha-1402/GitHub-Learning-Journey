## Useful Git Commands for GitHub

The following Git commands are commonly used while working with GitHub repositories:

- `git clone`
- `git status`
- `git add`
- `git commit`
- `git push`
- `git pull`
- `git branch`
- `git switch`
- `git checkout`
- `git merge`
- `git remote`
- `git fetch`
- `git diff`
- `git log`

These commands helped me understand the basic Git and GitHub workflow, including repository management, branching, collaboration, and version control.

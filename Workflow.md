## GitHub Workflow Diagram

```text
Create Repository
        ↓
Clone Repository
        ↓
Create or Modify Files
        ↓
git status
        ↓
git add .
        ↓
git commit -m "message"
        ↓
git push
        ↓
GitHub Repository Updated
```

## GitHub Workflow with Branches

```text
Create Repository
        ↓
Clone Repository
        ↓
Create Branch
        ↓
Switch Branch
        ↓
Make Changes
        ↓
git add .
        ↓
git commit
        ↓
git push
        ↓
Create Pull Request
        ↓
Merge Pull Request
        ↓
Delete Branch
```
